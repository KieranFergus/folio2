Universal Machine

A fast and accurate Universal Turing Machine Emulator - theoretically capable of simulating any other Turing machine hardware

Languages Used: C

Key Features:

  • - Has its own segmented memory and 8 registers
  • - Operates using 14 RISC-style instructions (arithmetic and logical operations, byte-oriented I/O, Memory management)

Lossy Image Compressor

A lightweight PPM image compressor/decompressor implemented via bitpacking of image pixels

Languages Used: C

Key Features:

  • - Compresses and image by taking its binary representation and bitpacking each 2x2 block of pixels into code strings
  • - Decompresses images by unpacking the strings into 2x2 blocks of pixels
